# SyncFlow 마켓 이미지 생성 프롬프트 (Google Stitch)

이 문서는 스토어 등록용 이미지 생성을 위해 Google Stitch에 바로 붙여넣을 수 있는 프롬프트를 제공합니다.
요구 방향: 심플, 밝고 깔끔한 앱 아이콘 스타일, 강한 가독성, 낮은 복잡도.

## 1) 앱 아이콘 프롬프트 (1024x1024, 심플/밝은 앱 아이콘 스타일)

```text
Create a clean and friendly app icon for a real-time collaborative Kanban board app.
Core concept: Kanban board itself (columns + cards), expressed with only essential shapes.
Visual: one clear board panel with 3 vertical columns and exactly 3 large rounded card blocks, arranged horizontally with slight overlap like a simple Gantt-chart sequence.
Style: simple modern mobile icon style, bright, soft, and tidy; smooth rounded geometry; minimal detail.
Color: fresh limited palette (3-4 colors max): light neutral background plus three card colors (sky blue, mint, soft yellow), and gentle gray for secondary lines.
Composition: centered, large main symbol, slightly increased vertical spacing between elements, high readability at small size.
Texture/lighting: subtle soft shading only; clean finish.
Negative constraints: no avatars, no people, no arrows, no busy UI, no dense details, no text, no watermark.
Output constraints: 1024x1024 PNG, full-bleed square, NO rounded corners, NO transparency/alpha, NO watermark, NO mockup frame.
```

## 2) 마케팅 이미지 프롬프트 (1024x500, 심플/밝은 앱 스타일)

```text
Create a clean and bright marketing hero image for a mobile app store listing.
Canvas: 1024x500.
Theme: real-time collaborative Kanban, focused on board structure and card clarity.
Scene: one large Kanban board with 3 clear columns and several card blocks, arranged to emphasize drag-and-drop organization.
Style: simple modern app illustration style, bright, soft, minimal, and tidy.
Color system: fresh limited palette (max 4 colors) with generous negative space.
Layout: strong center-left focal block, right side breathing room for potential copy.
Negative constraints: no avatars, no people, no arrows, no tiny UI details, no dense icons, no realistic device frame, no text, no watermark.
Output constraints: exact 1024x500 PNG, NO transparency/alpha, NO rounded frame.
```

## 빠르게 더 단순화할 때 추가 키워드
- `ultra minimal`
- `fewer elements`
- `large geometric blocks`
- `high-contrast silhouette`
- `strict 2D flat design`
