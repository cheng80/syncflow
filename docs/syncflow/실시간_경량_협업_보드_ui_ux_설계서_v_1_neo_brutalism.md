# 실시간 경량 협업 보드
## UI/UX 설계서 v1 (Neo-Brutalism 컨셉)

본 문서는 모바일 세로 1컬럼 구조 기준 UI/UX 설계 정의 문서이다.

---

# 1. 디자인 컨셉: Neo-Brutalism

## 1.1 핵심 방향
- 강한 대비(Color Contrast)
- 두꺼운 보더(2~3px 이상)
- 그림자 대신 명확한 오프셋 쉐도우
- 둥글지 않은 카드(또는 매우 약한 radius)
- 기능이 드러나는 직관적 인터페이스

## 1.2 디자인 원칙
1. 장식 최소화
2. 정보 우선
3. 동작이 명확하게 보이도록
4. 인터랙션은 과감하고 즉각적으로

---

# 2. 전체 화면 구조 (모바일 세로 기준)

앱은 아래 3단계 구조를 따른다.

1. 로그인 화면
2. 보드 선택(대시보드) 화면
3. 보드 상세 화면

---

# 3. 로그인 화면

## 3.1 구성
- 로고 또는 앱 타이틀 (Bold, Neo-Brutalism 타이포)
- 이메일 입력 필드 (두꺼운 보더)
- 인증 코드 입력 필드
- 로그인/인증 버튼 (강한 단색 배경 + 두꺼운 보더)

## 3.2 UX
- 이메일 입력 → 코드 발송
- 코드 입력 완료 시 자동 로그인
- 세션 유효 시 자동 보드 선택 화면 진입

---

# 4. 보드 선택(대시보드) 화면

## 4.1 목적
사용자가 참여 중인 보드 목록을 확인하고 선택하는 진입 화면

## 4.2 레이아웃 (MVP 단순화)
- 상단: 앱 타이틀 + 사용자 아바타
- 본문: 단순 보드 리스트 (ListView)
  - 보드 제목
  - 최근 수정 시각 (선택)
- 하단: + 새 보드 생성 버튼

> Neo-Brutalism 강한 카드 스타일은 2차 확장으로 미룸.
> 1차는 구현 속도 우선.

## 4.3 인터랙션
- 보드 탭 → 보드 상세 화면 진입
- 멤버 초대는 보드 상세 화면에서만 가능

---

# 5. 보드 상세 화면
 보드 상세 화면

- 상단 AppBar
  - 보드 제목
  - 우측 Presence 아바타

- 본문 영역
  - PageView 기반 1컬럼 구조
  - 세로 스크롤 카드 리스트

- 하단
  - 컬럼 내부 + Add Card 버튼

---

# 3. 컬럼 UI

## 3.1 컬럼 헤더
- 배경: 고채도 단색 (템플릿 기반)
- 보더: 3px 검정
- 텍스트: 굵은 폰트
- 카드 수 표시 (예: TODO (12))

## 3.2 컬럼 이동
- 좌우 스와이프
- 상단에 현재 컬럼 이름 고정 표시

---

# 4. 카드 UI

## 4.1 기본 카드 스타일
- 흰색 배경
- 3px 검정 보더
- 오프셋 그림자 (예: 6px x 6px 검정)
- Radius: 6~8px 이하 (또는 0)

## 4.2 카드 구성
- 상단: Title (Bold)
- 하단: Description 프리뷰 2줄
- 우측 상단: 우선순위 Dot
- 우측 하단: 이동 아이콘 / 드래그 핸들

---

# 5. 카드 인터랙션

## 5.1 같은 컬럼 내 이동 (재정렬)
- ≡ 핸들 길게 누르고 세로 드래그
- 드래그 중 카드 확대 + 그림자 강조
- 드롭 시 즉시 위치 반영 (optimistic)

## 5.2 컬럼 이동 모드

### 진입
- 카드 우측 이동 아이콘 탭

### 상태
- 상단에 "이동 모드" 배지 표시
- 선택된 카드 강조 표시
- 좌우 스와이프로 컬럼 선택

### 완료
- "이동 완료" 버튼 탭
- 선택된 컬럼 맨 위에 삽입

### 취소
- "취소" 버튼 또는 뒤로가기

---

# 6. Soft Lock UX

## 6.1 Description 편집
- 잠긴 카드 상세는 입력 비활성
- 상단에 굵은 락 배지 표시
- "OO님이 편집 중" 텍스트 강조

## 6.2 Owner Lock
- 카드 상단에 🔒 아이콘
- 배경 색상 약간 변경
- 수정 버튼 비활성화

---

# 7. Presence UI

- 상단 우측에 최대 3명 표시
- 초과 시 +N
- 탭 시 BottomSheet로 전체 목록

---

# 8. 컬러 시스템 (Neo-Brutalism 가이드)

## 8.1 기본 색상
- Primary: 강한 블루 / 레드 / 옐로우
- Background: #F5F5F5 또는 순백
- Border: #000000

## 8.2 상태 색상
- active: 기본
- done: 강한 그린 배지
- archived: 회색 처리

---

# 9. 애니메이션 가이드

- 이동 애니메이션은 빠르게 (200ms)
- 카드 드래그 시 scale 1.03~1.05
- 상태 변화는 즉각적 (지연 최소)

---

# 10. 2차 확장 대비

- 태블릿 가로 모드: 2~3컬럼 동시 표시
- 데스크탑: 멀티컬럼 보드
- 컬럼별 컬러 커스터마이징
